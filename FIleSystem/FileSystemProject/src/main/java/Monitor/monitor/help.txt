Если вы читаете это, значит вам потребовалась помощь в работе с нашей прекрасной файловой системой!
Ниже предоставлен перечень команд, которые могут быть вами использованы:
create -- команда создания новой файловой системы. После вызова команды будет предложено сохранение текущей файловой системы(см. save).
download -- команда загрузки файловой системы из файла. После вызова команды будет предложено сохранение текущей файловой системы(см. save).
file -- команда создания фала. После вызова команды программа попросит вас ввести данные для создания нового файла.
save -- команда сохранения текущей файловой системы. После вызова команды программа попросит вас ввести данные файла для сохранения.
add -- команда для добавления информации в файл. После вызова команды программа попросит вас ввести данные о файле для добавления информации.
delete -- команда удаления файла. После вызова команды программа попросит вас ввести данные для удаления некоторого файла.
exit -- команда завершения работы. После вызова команды будет предложено сохранение текущей файловой системы(см. save).
help -- помощь.
